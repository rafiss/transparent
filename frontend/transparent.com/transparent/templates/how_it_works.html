Transparent.How it Works

Product Search

Transparent searches through a number of different retail sites for the current lowest product prices. The searchable retailers are determined by the availability of a website parsing module for that retailer. During searches, the user can select which retailers to search through and the parsing algorithms to employ by selecting which modules to use.



Price Tracking

The product page for each item lists the current lowest price and the various attributes of the product. {% comment %} It also contains a price history graph to inform the user about the item's price trends. {% endcomment %} The history of the module and retailer through which the cheapest priced item was found is also viewable from the graph.
